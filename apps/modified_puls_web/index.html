What routing does

A reservoir or basin holds water back and lets it out through its outlet. The result is a routed outflow hydrograph that is attenuated (lower peak) and lagged (later peak) compared to the inflow. Drag the storage multiplier and watch both change.

The arithmetic HEC-HMS hides

Each timestep solves one equation against the storage-indication curve. Step through it: the green marker is exactly where this step's value is read off the curve.

☞ The curve is the routing. HEC-HMS just runs this loop for every timestep. On long events the stepper shows representative steps (first, peak, last, and evenly-spaced points) — each row is still an exact routed timestep.
